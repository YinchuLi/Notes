3. 笔记

	使用sphinx进行大段内容（书、专题）的整理。

		使用现有的模版，主要修改conf.py文件。使用reStructuredText格式写文档。
		可以更换模版，使用下列命令生成静态文件：
			sphinx-build -b html ./ builddir
		
		在Ubuntu下可以生成pdf电子书文档。

	使用wiki整理笔记。
	
		考虑使用vimwiki。
