##Coursera课程Human-Computer Interaction

加利福尼亚大学圣迭戈分校在[Coursera](https://class.coursera.org/hci-004)上的一门课程。在这门课中，你会学到如何设计技术使人愉快而不是沮丧。你会学到一些技术进行快速原型和多个界面评估，以及为什么快速成形和比较对于界面设计很重要。你会学到如何调查帮助你进行设计。
